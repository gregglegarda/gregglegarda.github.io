Alien Attack


This game was created using a programming language called Java and I used Jframe, Jpanel, and Jbuttons to create the User Interface. I used basic programming techniques to put entities as Objects and Classes. I also used a Unified Modeling Language (UML) to create a diagram of the classes and their specific functions.

3D Environment Rendering


This 3D environment was rendered using OpenGL and C++. The specific techniques used are Texture Mapping, Backface Removal, Illumination, and Camera Movements using keyboard inputs.

Linear Classification


This was created using Python. The program uses Machine Learning Algorithms to classify inputs based on their targets (Supervised Learning). The program draws a straight line to separate and correctly classify the two classes (as long as they are linearly separable).

Function Approximation


This was created using Python. The program uses Machine Learning algorithms to approximate a given function. The blue dotted line represents the given function and the red line represents the machine. The red line is initialized using random numbers and eventually will approximate the function given.

Facial Recognition


This Artificial Intelligence Application was created using Python. The program detects and recognizes faces in the webcam feed by using Haar Cascades and Histogram of Gradients (HOG) algorithims in the background. The program also implements Web Scraping to pull information from Instagram. The program needs to be trained for a specific face prior to recognition. This was a collaboration done with Cristina Giraldo.
